GUIA RAPIDA PARA ESTUDIANTES - GITHUB

Objetivo: descargar los archivos de la clase desde GitHub.

PRIMERA VEZ: DESCARGAR EL REPOSITORIO

1. Entrar a la carpeta de laboratorios:

cd ~/labs/devnet-src

2. Descargar el repositorio de la clase:

git clone https://github.com/ChaconFrancoLuis/CCNA-Automation-0405.git

3. Entrar a la carpeta del repositorio:

cd CCNA-Automation-0405

4. Ver los archivos:

ls

PARA ACTUALIZAR DESPUES

Cuando el profesor suba nuevos archivos, ejecutar:

cd ~/labs/devnet-src/CCNA-Automation-0405
git pull

EXPLICACION SIMPLE

- git clone = descargar la carpeta de la clase por primera vez.
- git pull = actualizar la carpeta cuando el profesor suba nuevos archivos.
- Los estudiantes no necesitan token, SSH key, commit, ni push.
